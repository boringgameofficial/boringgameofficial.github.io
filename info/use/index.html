愛用

Last Update: 2026-04-29

啟發自 Wiwi 現在愛用什麼和 Jerry 的平常使用的工具。

老實說，這個頁面原本只是寫給自己看的，但分享給你們好像也不會怎樣，所以我就直接公開吧！

免責聲明

以下列表僅供參考，可能有打錯的地方，麻煩大家校對。

💾 作業系統

  • Linux Mint - 入門款作業系統，也是我目前主要使用的作業系統
  • LMDE - 不依賴 Ubuntu 的 Linux Mint
  • Tails - 極具隱私的作業系統
  • Android - 以後換手機時再換成 LineageOS 吧

🖥️ 電腦軟體

💾 系統

  • Cinnamon - Linux Mint 預設的輕量桌面環境
  • awesome - 易用的視窗管理器（目前正在試用中）
  • Timeshift - Linux Mint 預設的系統備份
  • Web Apps - Linux Mint 預設的軟體，能將網站當應用程式來運作
  • GNOME Terminal - Linux Mint 預設的終端機
  • Font Manager - 字型管理器
  • Fcitx5 - 輸入法，我用的是新酷音

🔒 隱私

  • KeePassXC - 密碼管理器
  • oathtool -（終端機）2FA 驗證代碼檢視器
  • Proton VPN

📝 文件

  • micro -（終端機）文字編輯器
  • Obsidian -（專有軟體）Markdown 編輯器，目前只有編輯表格時會用
  • VSCodium - 去掉微軟追蹤的 VSCode
  • LibreOffice -（MPL v2.0 授權）Linux Mint 預設的 Office
  • Pandoc - 文件格式轉檔工具
  • XReader - Linux Mint 預設的文件檢視器

🎨 美工繪圖

  • Krita - 主要的繪圖軟體
  • MyPaint - 基礎的繪圖軟體
  • Flameshot - 螢幕截圖軟體
  • nomacs - 圖片編輯 / 檢視軟體

🎞️ 影音

  • Shutcut - 影片剪輯軟體
  • Open Broadcaster Software - 螢幕錄影 / 直播軟體
  • FFmpeg -（終端機）影片、音頻轉檔軟體
  • yt-dlp -（終端機、Unlicense license 授權）影片 / 音頻下載軟體
  • VLC Player - 影片 / 音頻播放軟體
  • FreeTube - 相當於沒有廣告的 YouTube
  • MuseScore Studio - 打譜軟體
  • Muse Sound Manager - 只是為了下 Muse Sound 用
  • pixitracker - 取樣機
  • cmus -（終端機）音樂播放軟體

📟 工具

  • ranger -（終端機）檔案管理器
  • ExifTool -（終端機）EXIF 編輯器
  • adb -（終端機）Android 裝置除錯工具
  • scrcpy - Android 裝置螢幕顯示 / 控制器
  • VirtualBox - 虛擬機
  • Fluent Reader -（BSD 3-Clause License 授權）RSS 閱讀器
  • Cubic - ISO 編輯器（限 Ubuntu 類）
  • Ventoy - 可開機 USB 隨身碟製作工具

🏡 自架

  • Docker
  • Ollama -（終端機）語言模型離線執行軟體
  • AnythingLLM - 我把它當作大型語言模型介面
  • Syncthing -（MPL v2.0 授權）檔案同步軟體
  • SearXNG - 自架搜尋引擎
  • Hugo - Markdown 架站

🌐 瀏覽器

  • LibreWolf -（MPL v2.0 授權）主要瀏覽器
  • Brave -（MPL v2.0 授權）次要瀏覽器
  • Tor Browser -（MPL 授權）最具隱私性的瀏覽器

🚀 瀏覽器擴充套件

Librewolf
  • CanvasBlocker -（MPL v2.0 授權）指紋追縱防止器
  • ClearURLs - 複製網址時清除追蹤元素
  • Disconnect - 廣告、指紋等類別的追蹤封鎖器
  • Equicord Web - Vencord 分支，Discord 客戶端模組
  • Firefox Color - 自定義瀏覽器主題
  • History Export
  • Kiwix JS Browser Extension - 離線 ZIM 檢視器（離線瀏覽器）
  • LibRedirect - 網站重新導向器
  • TWP - Translate Web Pages -（MPL v2.0 授權）網頁翻譯
  • uBlock Origin - 廣告攔截器
  • User-Agent Switcher and Manager
  • Vimium - 使用快速鍵瀏覽網頁
Brave
  • ClearURLs
  • Disconnect
  • Return YouTube Dislike - YouTube 倒讚顯示器
  • Simple Translate -（MPL v2.0 授權）網頁翻譯
  • SponsorBlock
  • uBlock Origin
  • User-Agent Switcher and Manager
  • Vencord Web - Discord 客戶端模組（網頁版）

💬 通訊軟體

  • Discord -（專有軟體）主要用的群組聊天軟體。我其實只想用網頁版，但網頁版不支援分享螢幕的同時傳輸內部音訊。
  • Vencord

🎮 遊戲

  • Shattered Pixel Dungeon - 地下城探險遊戲
  • Osu! - 社群向音遊兼製譜器
  • Arcade-plus - Arcaea 製譜器

  • Aurora Store - Google Play 非官方客戶端
  • Obtainium - 軟體更新抓取器
