我的個人網站發展史

2026-01-18

在深挖硬碟的時候，我找回了我的第一版個人網站。因為不知道要寫什麼當作 blog 的內容，乾脆談談我的個人網站發展史吧！

第一版

2022 年的某日，我心血來潮，想學習如何寫網頁，於是我找了 HTML 和 CSS 的教學：

然後寫出了很醜的網頁。

（如果你想要來看整個網頁長什麼模樣，可以點這裡。不過內容不是當初那個樣子，而是被改成和現在主頁相同的文字了。）

就網頁的介面而言，其實可以看出我當時在嘗試把那兩部影片的所有教學內容都加入到網頁中。

網頁檔案寫完之後，我將它們放在 Google Drive 上，用 DriveToWeb 去架網站。

第二版

2023 年 4 月 3 日，由於我覺得第一版的網頁很醜，再加上發現 Google 有協作平台，我試玩了一下，架了第二版的網頁。

（同樣地，內容也被改成和現在主頁相同的文字了。除外，字型也不一定符合當初的樣子。）

第三版

就是現在這一版。

和前兩版不同，我使用了 Hugo 和 Github Pages 架設了本網站，而我選的 Hugo theme 是 Hugo Bear Blog。後來我改動了 CSS，將背景加上圖片，文字區塊加上模糊效果。

這版也有深色模式。

至於為什麼改用 Hugo，其實是為了要 deGoogle。
